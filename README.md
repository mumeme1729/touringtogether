# toringtogether
### 説明
ツーリング仲間を目的地、日付から募集・検索できるアプリです。

### 使用技術
##### バックエンド
    ・Django
 VPS サービスconohaを使用  
 BD:Postgres
##### フロントエンド
  　・TypeScript
    ・React
    ・Redux ToolKit
